# hui-constraints

Project-level coding standards and AI behavior rules.

## Purpose

Keeps AI focused, accurate, and efficient by enforcing:

- **Coding standards**: style, quality, completeness
- **Modification rules**: scope control, confirmation workflows
- **Response formats**: error reports, optimizations, architecture
- **Forbidden patterns**: fabrication, security risks, scope creep

## Usage

Constraints are automatically active for technical tasks. View summary:

```
/constraints
```
