HUI

Concise technical communication.

HUI is a writing-style toolkit for coding agents. It removes conversational filler while retaining technical terms, code, exact errors, and safety guidance.

Commands

Enable persistent concise-output mode. Use lite, full, ultra, or Wenyan variants.

/hui demo

Show fixed 本地文本示例. No model call and no local state write.

/hui-compress

Rewrite supported natural-language files with backups and structural validation.

/hui-stats

Show observed local session usage from Claude Code logs. No savings or cost estimates.

Boundaries

Clarity wins when compression would mislead.

HUI uses normal prose for security warnings, irreversible operations, and ambiguous multi-step instructions. Code, commits, and PR descriptions remain normal.
